== Description ==

This plugin's widget allows you to auto generate a QR Code for your web pages. We created this to allow websites to provide visitors the ability to easily continue reading websites on the run! 

TakeItMobile is developed by NOVA Advertising, http://www.novaadvertising.com

TakeItMobile uses the technology of QR-SERVER, read more here: http://goqr.me/


== Installation ==

1. Upload `nova_takeitmobile.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the widget like any other widget.
